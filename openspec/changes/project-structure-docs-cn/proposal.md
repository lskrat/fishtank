## Why

当前项目缺乏一份全面且深入的中文指南，用于解释项目结构、核心依赖关系、上手流程以及配置修改方法。这使得中文开发者在理解和贡献项目时面临较高的门槛。为了降低学习成本，提高开发效率，需要一份详尽的中文项目结构文档。

## What Changes

将创建一个新的中文文档（例如 `docs/zh-CN/project-structure.md`），包含以下内容：
- **项目结构概览**：详细说明各个目录和关键文件的作用。
- **依赖关系分析**：解释主要模块之间的相互依赖和调用关系。
- **从头了解项目**：提供一个逻辑清晰的阅读和理解代码的路径。
- **核心代码片段**：展示关键功能的代码实现，并提供跳转到源码的链接。
- **配置修改指南**：说明常见配置项的位置和修改方法。
